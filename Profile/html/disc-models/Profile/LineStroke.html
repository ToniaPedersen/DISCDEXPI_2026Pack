DashArray¶

Data Property

A pattern of gaps in millimeters, based on the same principle as the SVG attribute stroke-dasharray.

  • An empty DashArray corresponds to a solid line without any gaps.

  • Other than in SVG, a DashArray must contain an even number of values.

Type: Double

Multiplicity: *

Ordered: yes

Unique: no

LateralOffset¶

Data Property

A lateral offset in millimeters of the stroke, i.e., an offset perpendicular to the drawing direction.

  • If LateralOffset is zero, the line is drawn without any offset.

  • If LateralOffset is positive, the line is drawn with an offset to the right w.r.t. the drawing direction.

  • If LateralOffset is negative, the line is drawn with an offset to the left w.r.t. the drawing direction.

A typical application of LateralOffset could be to define an AggregatedStroke for a pipe with heat tracing:

  • The first LineStroke, corresponding to the actual pipe, would have LateralOffset zero.

  • The second LineStroke, corresponding to a dashed heat tracing line, would have a non-zero LateralOffset.

Type: Double

Multiplicity: 1

Ordered: no

Unique: no

LineRounding¶

Data Property

The appearance of the end of a line or a line section. This attribute corresponds to the SVG attributes stroke-linecap and stroke-linejoin:

  • The value Butt corresponds to stroke-linecap=”butt” and stroke-linejoin=”miter”.

  • The value Round corresponds to stroke-linecap=”round” and stroke-linejoin=”round”.

Type: LineRounding

Multiplicity: 1

Ordered: no

Unique: no
